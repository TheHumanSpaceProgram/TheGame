To run the game, simply run XtremeStackingTheRaceToSpace.exe

To build the project from scratch, you will need the program Unity and a program to unzip .zip files.
Then you need to
 -	Unzip HumanStackingTheRaceToSpace.zip
 -	Open the folder HumanStackingTheRaceToSpace with unity
 -	Press File -> Build & Run
